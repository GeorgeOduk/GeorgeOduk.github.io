Global Layoffs Exploratory Data Analysis

A MySQL exploratory data analysis project investigating global layoffs by company, industry, country, year, month and company stage.

Project Summary

This project demonstrates how MySQL can be used to explore and understand a cleaned global layoffs dataset before dashboarding or deeper analysis. The analysis focused on identifying layoff patterns across companies, industries, countries, years, months and company stages.

The aim was to move beyond basic querying by using SQL as an analytical tool for data profiling, trend analysis, ranking and early insight generation. The project includes company-level totals, industry and country breakdowns, yearly and monthly trends, rolling cumulative totals and top company rankings by year.

This project strengthens my portfolio by showing that I can investigate a dataset independently, ask structured analytical questions and produce SQL outputs that could support future reporting, dashboarding or business decision-making.

Tools Used

  • MySQL for exploratory data analysis and query-based investigation
  • MySQL Workbench for writing and running SQL queries
  • Aggregate functions for summary metrics and pattern identification
  • GROUP BY and ORDER BY for category comparison and ranking
  • Date functions for yearly and monthly trend analysis
  • CTEs for structuring multi-step analytical queries
  • Window functions for rolling totals and ranking analysis
  • DENSE_RANK() for identifying top companies by layoffs per year
  • Query outputs for documenting findings and supporting interpretation

SQL Analysis Preview

Exploratory SQL Analysis

Uses MySQL queries to profile the cleaned layoffs dataset, summarise key measures, compare segments and identify patterns that could inform further reporting, dashboarding or business recommendations.

Analysis Approach

  • Reviewed the cleaned dataset structure, key fields and available measures
  • Identified maximum layoff values and percentage laid off values
  • Analysed companies with 100% workforce layoffs
  • Calculated total layoffs by company
  • Reviewed the date range covered by the dataset
  • Grouped layoffs by industry to identify heavily affected sectors
  • Grouped layoffs by country to identify geographic concentration
  • Analysed layoffs by year to understand wider time-based patterns
  • Analysed layoffs by company stage to compare business maturity levels
  • Created monthly layoff trend outputs
  • Calculated rolling cumulative monthly layoffs using window functions
  • Ranked the top companies by layoffs for each year
